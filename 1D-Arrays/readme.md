## Едномерни Масиви
---
Деф: Структура от данни с фиксиран брой елементи от един и същи тип, съхранявани в последователни адреси в паметта. Достъпът до всеки елемент става чрез индекс.

---
### Статични едномерни масиви
„Статичен“ означава, че размерът на масива е фиксиран (константен/непроменим - от там и думата статичен) и е известен на компилатора ощв по време на компилация
``` cpp
const unsigned CONSTANT_VALUE = 5;

int arr[CONSTANT_VALUE] {0}; // {0} - всички елементи в масива са инициализирани със стойност 0
arr[0] = 6;
arr[4] = 5

```
&nbsp;&nbsp;[0]&nbsp;&nbsp;&nbsp;&nbsp;[1]&nbsp;&nbsp;&nbsp;&nbsp;[2]&nbsp;&nbsp;&nbsp;&nbsp;[3]&nbsp;&nbsp;&nbsp;&nbsp;[4]&nbsp;&nbsp;
| 6  | 0 | 0  | 0  | 5  | 
|----|---|----|----|----|

Адресите на елементите са последователни:
```
0x1000   0x1004   0x1008   0x100C   0x1010
  |        |        |        |        |
 [6]      [0]      [0]      [0]      [5]
```

(4 байта на елемент -> int)
